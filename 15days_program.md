# Extended Program – Agentic AI & Enterprise Transformation (15 Days)

## Program Vision
This extended format provides **doctoral‑level depth** across 15 days (8 hours/day). Each of the 5 core themes is expanded into **3 sub‑days** with deeper technical labs, enterprise case studies, and research assignments. Participants will master **Agentic AI, DevOps, MLOps, FinOps, AIOps, Governance, and TOGAF ADM** end‑to‑end.

---

## Module 1 – Foundations (Days 1–3)

### Day 1 – Python & APIs
- Python essentials for AI (functions, JSON, APIs)  
- FastAPI + Uvicorn microservices  
- Postman & Bruno API testing  
- Lab: Build a FastAPI microservice calling an LLM  

### Day 2 – LLM Architecture
- Tokens, embeddings, transformer layers  
- LLM providers: Hugging Face, Azure OpenAI, AWS Bedrock, Anthropic, Meta AI  
- Lab: Generate embeddings, semantic similarity search  

### Day 3 – RAG & VectorDBs
- RAG pipeline: ingestion, chunking, retrieval  
- VectorDBs: Pinecone, FAISS, Weaviate, Chroma  
- Lab: Build a RAG pipeline with FAISS + Pinecone  

---

## Module 2 – Orchestration & DevOps (Days 4–6)

### Day 4 – Agentic AI Concepts
- AI Agents vs Agentic AI  
- Planning & reasoning architectures  
- Secure tool use, guardrails  
- Lab: Build a reasoning agent with structured output  

### Day 5 – Orchestration Frameworks
- LangChain, LangGraph, CrewAI, AutoGen, Semantic Kernel  
- Retry/fallback logic  
- Lab: Multi‑agent orchestration pipeline  

### Day 6 – DevOps Integration
- Jenkins pipelines, Docker, Kubernetes  
- CI/CD automation for agent microservices  
- Lab: Deploy FastAPI agent service with Docker + Kubernetes
