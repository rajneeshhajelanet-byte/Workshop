
# Day 1 – Foundations of AI, LLMs & RAG

## Session Vision
Day 1 establishes the **foundational skills** required for Agentic AI. Participants will gain hands‑on mastery of **Python, APIs, embeddings, NLP pipelines, and Retrieval‑Augmented Generation (RAG)**. This day also introduces **due diligence frameworks** (pain points, As‑Is vs To‑Be, KPIs, risk assessment) to connect technical depth with enterprise adoption.

---

## Morning Session (4 Hours)

### 1. Python Essentials
- **[Python Basics](ca://s?q=Python_basics_for_AI)** – variables, functions, loops, JSON handling  
- **API Integration** – `requests`, `openai`, `boto3`  
- **FastAPI + Uvicorn** – building microservices for AI agents  
- **Postman & Bruno** – API testing and validation  

**Hands‑on Lab**:  
- Create a **FastAPI microservice** that calls an LLM API.  
- Test endpoints with Postman/Bruno.  

---

### 2. LLM Architecture
- **[Tokens](ca://s?q=LLM_tokens_explained)** – tokenization, context windows  
- **Embeddings** – semantic representation of text  
- **Transformer Layers** – attention mechanism, positional encoding  
- **LLM Providers** – Hugging Face, Azure OpenAI, AWS Bedrock, Anthropic, Meta AI  

**Hands‑on Lab**:  
- Generate embeddings using `sentence-transformers`.  
- Compare semantic similarity vs keyword search.  

---

## Afternoon Session (4 Hours)

### 3. Retrieval‑Augmented Generation (RAG)
- **[RAG Pipeline](ca://s?q=RAG_pipeline_basics)** – ingestion, chunking, embeddings, retrieval  
- **VectorDBs** – Pinecone, FAISS, Weaviate, Chroma  
- **Memory Design** – persistent vs ephemeral memory  
- **Token Management** – context optimization, truncation strategies  

**Hands‑on Lab**:  
- Build a **RAG pipeline**: ingest documents → embed → store in VectorDB → retrieve → generate answer.  
- Compare FAISS vs Pinecone retrieval performance.  

---

### 4. NLP & Document Intelligence
- **OCR/NLP Pipelines** – extracting text from PDFs/images  
- **Speech Recognition Frameworks** – Azure Cognitive Services, Whisper  
- **Conversational AI** – chat/voice interfaces with FastAPI + React.js  

**Hands‑on Lab**:  
- Process a PDF with OCR → embed text → query with RAG.  
- Build a simple **chatbot UI in React.js** connected to FastAPI backend.  

---

## Due Diligence Frameworks (Integrated Throughout Day 1)
- **Pain Points** – identify inefficiencies in current workflows  
- **Process Gap Analysis** – As‑Is vs To‑Be mapping  
- **KPIs** – define measurable outcomes (MTTR, containment rate, cost reduction)  
- **Risk Assessment** – compliance, bias, scalability, data privacy  
- **Questionnaires & Document Review** – stakeholder interviews, SOP analysis  

---

## Key Takeaways
By the end of Day 1, participants will:
- Master **Python + FastAPI** for AI microservices  
- Understand **LLM architecture, embeddings, and tokens**  
- Build a **working RAG pipeline** with VectorDBs  
- Apply **due diligence frameworks** to connect technical solutions with enterprise pain points  
- Prepare for deeper orchestration and governance in subsequent days
